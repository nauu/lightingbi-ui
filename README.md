# lightingbi-ui
The ui for [Intelligent analysis system by Rust](https://github.com/nauu/lightingbi), base on [Ant Design Pro](https://pro.ant.design).

## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
